## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/yourusername/falzthebahdguy_countdown.git
   cd falzthebahdguy_countdown

   ```

2. **Install dependencies:**

   ```bash
   flutter pub get

   ```

3. **Run the app:**
   ```bash
   flutter run
   ```
